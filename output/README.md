# yq

A portable command-line YAML, JSON, XML, CSV, TOML and properties processor —
a lightweight, dependency-free Go binary using jq-like syntax.

- Upstream: https://github.com/mikefarah/yq
- Documentation: https://mikefarah.gitbook.io/yq/
- Packaging: https://github.com/dariogriffo/yq-debian
- Repository: https://deb.griffo.io
